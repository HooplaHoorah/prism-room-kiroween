# Changelog

All notable changes to this project will be documented in this file.
This project loosely follows the spirit of “Keep a Changelog”.

## [1.3.9] — 2025-11-07
### Added
- **Settings modal** with Photosensitive Mode, Hints, and Audio toggle.
- **Sticky event banner** with **Replay Last** button.
- **Seeded runs** via `?seed=<int>` for reproducible sessions.
- **HUD polish** (readability improvements, hotkeys) and inventory label.
- **Accessibility**: ARIA live region for events; full keyboard path.

### Notes
- This is the **Kiroween Kernel demo** release.
- See the README for usage and known limitations.
